# f3:point_up::v::raised_hand:
Flip a card written by Five Fingers. Recommended for use in online agile meeting:black_joker:

## Getting Started:rocket:

```zsh
$ yarn install
$ yarn dev
```

- frontend: 3000
- backend : 5000
